# ikwukao

Modern Backend Engineer portfolio website built with Next.js and Tailwind CSS.

## Overview

This project is a modern developer portfolio designed to showcase:

- Backend engineering skills
- AI and automation projects
- Technical experience
- Professional branding
- Contact information and social links

The website focuses on a clean cyber-minimal aesthetic with responsive design, smooth interactions, and optimized performance.

### Installation

Clone the repository:

```bash
git clone https://github.com/ikwukao/ikwukao.github.io
```

Navigate into the project directory:

```bash
cd ikwukao
```

Install dependencies:

```bash
npm install
```

---

### Running the Development Server

Start the local development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

### Build for Production

Generate a production build:

```bash
npm run build
```

Run production server:

```bash
npm start
```

---

#### Deployment

2. Import repository into Vercel
3. Deploy instantly
